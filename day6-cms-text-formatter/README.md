\# CMS Text Formatter



\## Use case

Used by digital agencies and content teams to clean raw text before publishing into CMS platforms like WordPress, Webflow, or internal editors.



\## What problem it solves

Cleans and normalizes raw text pasted into CMS editors. Fixes spacing, punctuation, quotes, and junk characters while preserving paragraphs.



\## Who itâ€™s for

CMS editors, content writers, bloggers, and anyone pasting raw or unformatted text into publishing systems.



\## Features

\- Normalizes multiple spaces while preserving paragraph breaks

\- Cleans repeated punctuation into a single character

\- Removes junk special characters while preserving valid ones

\- Applies correct opening and closing single and double quotes

\- Trims leading and trailing spaces from each line



\## How to run

1\. Download or clone the repository

2\. Open `index.html` in a browser
