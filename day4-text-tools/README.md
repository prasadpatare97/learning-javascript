# Text Tool

## What problem it solves
Counts characters, words, and sentences in text. Cleans up messy text so users don’t have to manually fix casing and spacing.

## Who it’s for
Anyone who works with text and needs quick cleanup and basic counts, such as content writers, CMS editors, students, and social media managers.

## Features
- Live character, word, and sentence count
- Converts text to lowercase, uppercase, or title case
- Capitalizes sentences in messy paragraphs
- Normalizes irregular spacing

## How to run
1. Download or clone the repository
2. Open `index.html` in a browser
