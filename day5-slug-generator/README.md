\# Slug Generator



\## What problem it solves

Converts text into clean, URL-safe slugs. Removes special characters and normalizes spacing so titles can be safely used in URLs.



\## Who itâ€™s for

Anyone creating URLs from text, such as content writers, CMS editors, bloggers, and developers.



\## Features

\- Converts text to lowercase

\- Removes special characters, keeping only letters and numbers

\- Normalizes multiple spaces into a single dash

\- Collapses multiple dashes into one

\- Generates URL-safe slugs from messy input

1\. Download or clone the repository

2\. Open `index.html` in a browser
